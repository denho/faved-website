Add a link and Faved does the rest — pulling in the title, description, and preview image automatically. It also keeps those details fresh over time, so your
collection never goes stale.

## Adding a bookmark

When you add a new link, Faved **autofills the page details** for you — it fetches the page title,
description, and preview image from the URL so you rarely have to type anything by hand. Add tags and
notes, then save.

> [!NOTE]
> Faved detects duplicate bookmarks as you save them, warning you before a link you already have gets
> added again — so duplicates never quietly pile up.

For one-click saving while browsing the web, install the
[browser bookmarklet](/docs/getting-started/using-browser-bookmarklet) or set up
[Apple Shortcuts](/docs/getting-started/saving-with-apple-shortcut). Already have links in another tool?
See [Importing bookmarks](/docs/guides/importing-bookmarks).

## Editing and deleting bookmarks

Update a bookmark's title, description, tags, or notes at any time, or delete it when you no longer need
it. To work across many bookmarks at once — retagging in bulk or clearing out links you don't want — use
[bulk actions](/docs/guides/bulk-actions).

## Refetching metadata

If a page changes after you saved it — or a bookmark was [imported](/docs/guides/importing-bookmarks)
without complete information — you can **refetch its metadata** (title, description, and preview image)
in a single click directly from the list view, without opening each bookmark. To refresh many bookmarks
at once, use [bulk actions](/docs/guides/bulk-actions).

## Reliable preview images

Bookmark preview images are stored locally by Faved, so they keep loading reliably over time even if the
source site changes or removes the original image — no broken thumbnails.

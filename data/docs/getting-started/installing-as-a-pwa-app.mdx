Faved is a Progressive Web App, so you can install it directly on your desktop or mobile device and use
it like a native app. Installed, it opens faster, stays out of your browser clutter, and gives you a more
focused, app-like experience.

## Supported platforms and browsers

> [!NOTE]
> Even if you don't have a browser that supports PWA installation, you can still enjoy a great experience by accessing Faved as a regular
website and adding it as a bookmark for quick access.

#### Desktop


* Chrome, Edge, Opera (Chromium Browsers) on Windows, Mac, and Linux.
* Safari on macOS Sonoma (14) and later.


#### Mobile

* Chrome, Firefox, Edge, Opera, and Samsung Internet on Android.
* Safari on iOS/iPadOS.
* Chrome, Edge, and Firefox on iOS/iPadOS 16.4 and later.


## Chrome, Edge, and Opera on Windows, Mac, and Linux

Supported on Windows, macOS, and Linux. All three browsers show a dedicated install icon in the address
bar when a PWA is available.

<Steps>

    <Step title="Open Faved">

        Navigate to your Faved instance URL. If you're using the cloud version, visit
        [faved.cloud](https://faved.cloud).

    </Step>

    <Step title="Click the install icon">

        Click the **Install** icon on the right side of the address bar — it looks like a monitor with
        a download arrow.

        [//]: # (![Install icon on the right side of the Chrome address bar]&#40;/static/images/docs/pwa/desktop-install-icon.png&#41;)


        > [!TIP]
        > If you don't see the install icon, open the browser menu and look for **Cast, save and share → Install page as app** (Chrome) or **Apps → Install this site as an app** (Edge, Opera).

    </Step>

    <Step title="Confirm">

        Click **Install** in the popup that appears.

        [//]: # (![Chrome PWA install confirmation popup]&#40;/static/images/docs/pwa/desktop-confirm.png&#41;)

    </Step>

</Steps>

Depending on your platform, Faved will appear your taskbar, Dock, start menu, or app launcher.

## Safari on Mac

Requires macOS Sonoma (14) or later.

<Steps>

    <Step title="Open Faved in Safari">

        Navigate to your Faved instance URL. If you're using the cloud version, visit
        [faved.cloud](https://faved.cloud).

    </Step>

    <Step title="Choose File → Add to Dock">

        From the menu bar, choose **File → Add to Dock**.

        [//]: # (![Safari File menu with Add to Dock option highlighted]&#40;/static/images/docs/pwa/macos-safari-file-menu.png&#41;)

    </Step>

    <Step title="Confirm">

        [//]: # (![macOS Safari Add to Dock confirmation dialog]&#40;/static/images/docs/pwa/macos-safari-confirm.png&#41;)

    </Step>

</Steps>

Faved will appear in your Dock and opens as a standalone window separate from Safari.

## Android

Supported in Chrome, Firefox, Edge, Opera, and Samsung Internet. The steps below use Chrome, but the
flow is similar in other supported browsers.

<Steps>

    <Step title="Open Faved in your browser">

        Navigate to your Faved instance URL. If you're using the cloud version, visit
        [faved.cloud](https://faved.cloud).

    </Step>

    <Step title="Open the browser menu">

        Tap the **three-dot menu** icon in the top-right corner.

        [//]: # (![Chrome three-dot menu icon on Android]&#40;/static/images/docs/pwa/android-menu.png&#41;)

    </Step>

    <Step title="Tap Add to Home Screen">

        Select **Add to Home Screen**. If the browser has already detected the PWA, you may instead
        see an **Install app** banner at the bottom of the screen — tap that instead.

        [//]: # (![Add to Home Screen option in Chrome's Android menu]&#40;/static/images/docs/pwa/android-add-to-home-screen.png&#41;)

    </Step>

    <Step title="Confirm">

        Tap **Add** in the confirmation dialog.

        [//]: # (![Android install confirmation dialog]&#40;/static/images/docs/pwa/android-confirm.png&#41;)

    </Step>

</Steps>

Faved will appear on your home screen and app drawer.

## iOS and iPadOS

On iOS and iPadOS **16.4 and later**, installation is supported in Safari, Chrome, Edge, and Firefox.
On **16.3 and earlier**, only Safari supports PWA installation — switch to Safari if you're on an
older version.

<Steps>

    <Step title="Open Faved in your browser">

        Navigate to your Faved instance URL. If you're using the cloud version, visit
        [faved.cloud](https://faved.cloud).

    </Step>

    <Step title="Tap the Share button">

        Tap the **Share** button (the box with an arrow pointing up). In Safari it's at the bottom of
        the screen; in other browsers it's typically in the address bar or the browser menu.

        [//]: # (![Share button in Safari at the bottom of the screen]&#40;/static/images/docs/pwa/ios-share-button.png&#41;)

    </Step>

    <Step title="Tap Add to Home Screen">

        Scroll down in the share sheet and tap **Add to Home Screen**.

        [//]: # (![Add to Home Screen option highlighted in the iOS share)

[//]: # (sheet]&#40;/static/images/docs/pwa/ios-add-to-home-screen.png&#41;)

    </Step>

    <Step title="Confirm">

        Optionally edit the name, then tap **Add** in the top-right corner.

        [//]: # (![iOS confirmation dialog with the Add button in the top-right corner]&#40;/static/images/docs/pwa/ios-confirm.png&#41;)

    </Step>

</Steps>

Faved will appear on your Home Screen and in the App Library, and opens as a full-screen app without
the browser toolbar.

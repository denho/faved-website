> [!NOTE]
> This guide applies to the self-hosted version of Faved only. [Faved Cloud](https://app.faved.to/signup) instances are updated automatically — you're always on the latest version with nothing to do.

Depending on whether you previously completed the [Quick Installation](/docs/getting-started/installation#quick-installation) or the [Advanced Installation](/docs/getting-started/installation#advanced-installation), the steps for updating Faved to the latest version will differ.

- **[Updating after Quick Installation](#updating-after-quick-installation)** — if you used the Quick Installation method.
- **[Updating after Advanced Installation](#updating-after-advanced-installation)** — if you used the Advanced Installation method with Docker Compose.

Choose the section that matches your setup below.

## Updating after Quick Installation

Follow these steps if you installed Faved using the [Quick Installation](/docs/getting-started/installation#quick-installation) method.

<Steps>

<Step title="Pull the latest stable image from Docker Hub">

```bash
docker pull denho/faved
```

</Step>

<Step title="Stop and remove the running container">

```bash
docker stop faved
docker rm faved
```

If you changed the container name during installation, replace `faved` with the name you used.

</Step>

<Step title="Start a new Docker container">

```bash
docker run -d --name faved -p 8080:80 -v faved-data:/var/www/html/storage denho/faved
```

Done! You now have the latest version running.

</Step>

</Steps>

## Updating after Advanced Installation

Follow these steps if you set up Faved using the [Advanced Installation](/docs/getting-started/installation#advanced-installation) method.

<Steps>

<Step title="Pull the latest `docker-compose.yml` file to your Faved directory">

Fetch the latest file from the [official repository](https://github.com/denho/faved/blob/main/docker-compose.yml):

<CodeTabs>
<CodeTab label="curl">
```bash
cd faved-app
rm docker-compose.yml
curl -O https://raw.githubusercontent.com/denho/faved/refs/heads/main/docker-compose.yml
```
</CodeTab>
<CodeTab label="wget">
```bash
cd faved-app
rm docker-compose.yml
wget https://raw.githubusercontent.com/denho/faved/refs/heads/main/docker-compose.yml
```
</CodeTab>
</CodeTabs>

</Step>

<Step title="Recreate the Docker service with the latest image">

```bash
docker compose up --pull always -d
```

Done! You now have the latest version running.

</Step>

</Steps>
